Geting Started with Aquafox

Ready to try Aquafox? Download the latest version here!
Aquafox is available for PowerPC Macs with a G3, G4-7400, G4-7450, or G5 processor, running Mac OS X 10.4.9 "Tiger" and later.

Installation:

To install Aquafox, simply mount the .dmg file and drag Aquafox to your Applications folder.

For those who feel more adventurous, you can compile Aquafox yourself using The Unofficial TenFourFox Development Toolkit. All released builds are compiled this way, so they will be identical. (Hint: You can specify any of the development branches to try the latest changes yourself!)
